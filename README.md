### 💕Announcement


新成员招募：

### 💕研究方向

（1）计量心理学：发展针对纵向数据和密集追踪数据的统计模型，包括算法开发、模型构建、应用拓展等。

（2）健康心理学：从动态系统的视角解释人的健康行为过程，包括饮食、情绪、压力管理等。
